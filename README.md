# Evaluating-Earthquakes
The main goal of the analysis was to gain some general statistical knowledge of earthquakes so that anyone reading the code could extract meaningful insights. 

The earthquake data in this project was found on Kaggle. The years included in this dataset spans from years 1995-2023. 

Attatched to this repository you can find the jupyter notebook that I analzed the earthquake data with. I included comments to help showcase some of my decision making for transparency reasons and also just general understandings. Attatched to this repository you can also find a pdf which contains a written conclusion of key-takeaways from the data. Arguably the best way of viewing the data would be to browse through the jupyter notebook. The code has been updated to be more easily understood.  66
