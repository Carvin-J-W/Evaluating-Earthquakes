# Evaluating-Earthquakes
The years included in this dataset spans from years 1995-2023. This project involves some basic data analysis of Earthquake data found on Kaggle. The main goal of the analysis was to gain some general statistical knowledge of Earthquakes so that anyone could make some informative conclusions.

Attatched to this repository you can find the jupyter notebook that I analzed the Earthquake data with. I included comments to help showcase some of my decision making for transparency reasons and also just general understandings. Attatched to this repository you can also find a pdf which contains a written conclusion of key-takeaways from the data.
